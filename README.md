# PetCare - Sistema de Gestão de Clínica Veterinária
## Descricao
O **PetCare** e um sistema de console, desenvolvido em java, para a gestão de uma clínica
## Funcionalidades
- **Proprietarios:** Cadastrar, listar, atualizar, excluir e buscar por CPF
- **Veterinarios:** Cadastrar, listar, atualizar, excluir e buscar por CRMV
- **Animais:** Cadastrar, listar, atualizar e excluir
- **Consultas:** Agendar, listar, atualizar e excluir

## Tecnologias Utilizadas

- Java 17+
- PostgreSQL
- JDBC

## Como Executar

1. Configure o arquivo `.env` com suas credenciais do banco
2. Compile e execute o projeto

## Seguranca

Este projeto usa variaveis de ambiente para proteger informacoes sensiveis. Copie o arquivo `.env.example` para `.env` e preencha com suas credenciais.
